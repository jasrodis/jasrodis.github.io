## Deployment

This site is hosted via [GitHub Pages](https://pages.github.com/).

Push to the `main` branch to deploy automatically.

## Custom Domain

To use a custom domain (e.g. `iasonrodis.com`), add a `CNAME` file with your domain name and configure DNS as described in the deployment guide.
